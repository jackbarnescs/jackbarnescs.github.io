1.4.2 Data Structures

Specification Points:

  • Arrays (of up to 3 dimensions), records, lists, tuples.
  • The following structures to store data: linked-list, graph (directed and undirected), stack, queue, tree, binary search tree, hash table.
  • How to create, traverse, add data to and remove data from the data structures mentioned above. (This can be either using arrays and procedural programming or an object-oriented approach).
